# Benchmark protocol-handler sources

The released benchmark binaries are built from the following open-source protocol stacks or distributions:

| Protocol | Upstream implementation | URL |
| --- | --- | --- |
| BACnet | BACnet Stack | https://github.com/bacnet-stack/bacnet-stack |
| CIP / EtherNet/IP | OpENer | https://github.com/EIPStackGroup/OpENer |
| IEC104 | lib60870 | https://github.com/mz-automation/lib60870 |
| MMS | libiec61850 | https://github.com/mz-automation/libiec61850 |
| Modbus | libmodbus | https://github.com/stephane/libmodbus |
| S7comm / Snap7 | Snap7 | https://sourceforge.net/projects/snap7/ |
